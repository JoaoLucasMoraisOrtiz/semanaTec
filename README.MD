PARA INICIAR O PROGRAMA É NECESSÁRIO ALGUMAS DEPENDÊNCIAS:
1º TKINTER: O tkinter é um compositor de janelas e ele foi utilizado nesta aplicação. Para instalar o tkinter, rode no seu terminal ou CMD o comando abaixo:

pip install tk
pip install tkhtmlview
pip install tkTimePicker

2º MYSQL: O banco de dados utilizado foi o mysql, portanto precisamos de seu conector para fazer a ligação:

pip install mysql-connector-python

3º PYTHON3 E MYSQL: tambem precisamos do python3 e do mysql. Para instala-los varia em cada sistema, você pode encontrar varios tutoriais sobre suas intalações na internet.

4º CRIAR O BANCO DE DADOS: importe o banco de dados semTecDb.Sql para o seu banco de dados

5º RENOMEIE E COLOQUE OS DADOS DO secrets.json.sample: apenas preencha os dados onde é solicitado e renomeie o arquivo para secrets.json
